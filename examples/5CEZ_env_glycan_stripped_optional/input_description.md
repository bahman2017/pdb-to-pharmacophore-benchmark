# 5CEZ — Viral Glycoprotein Surface-Mapping Demo (Glycan-Stripped)

## Overview

Viral protein **epitope-surface constraint mapping demo** using PDB `5CEZ`.

## Preprocessing

- Glycans, waters, and HETATM records stripped for an apo-style surface-mapping benchmark
- Protein ATOM records retained

## Context Metadata

- Temperature: 310 K
- pH: 7.4

## Public-Safe Claim

This example is **not** a vaccine design claim and does **not** imply immunogenicity, neutralization, therapeutic efficacy, or clinical relevance. It demonstrates export of explainable surface-adjacent anchor constraints from a viral glycoprotein structure under a documented context profile.
