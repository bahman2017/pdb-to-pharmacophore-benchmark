# 9TZD — KempTIM1 Retrospective Benchmark

## Overview

Retrospective benchmark on a public **de novo enzyme** structure (KempTIM1, PDB `9TZD`).

## Preprocessing

- Public deposited structure used as benchmark input
- Standard benchmark artifact export (constraints, plan, pharmacophore PDB, HTML viewer)

## Context Metadata

- Temperature: 310 K
- pH: 6.5
- Interpretation: **acidic-context stress test** for metadata-driven constraint export (not a tumor or clinical environment validation)

## Public-Safe Claim

This example demonstrates **context-aware, explainable constraint generation** on a de novo enzyme structure. It does not establish catalytic mechanism proof, therapeutic utility, or clinical relevance.
